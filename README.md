# Employee Benefits Claims Analysis
 
## Project Overview
This project analyzes employee benefits claim data to understand claim trends, department spending, and approval patterns. The analysis was performed using SQL, Python, and Tableau to generate insights and visualize claim behavior across departments and benefit types.
 
## Tools Used
- SQL
- Python (Pandas, Matplotlib)
- Tableau
- Excel
 
## Key Business Questions
- Which departments generate the highest claim cost?
- Which benefit types contribute the most to expenses?
- What is the distribution of claim approvals, rejections, and pending claims?
- How do employee claims change over time?
 
## Analysis Process
1. Data exploration and cleaning using Python (Pandas)
2. Aggregation and trend analysis
3. Visualization using Tableau
4. Dashboard development for business insights
 
## Dashboard Insights
The dashboard highlights:
- Total claim cost by department
- Benefit type cost comparison
- Claim approval distribution
- Monthly claim trends
